שאלות נפוצות

20 השאלות שמגיעות הכי הרבה. לא מצאת? שאל ב-GitHub.

האם השירות חינמי?
כן. לחלוטין. קוד פתוח Apache 2.0.
איך זה שונה מאפליקציית פיקוד העורף?
השירות הוא משלים בלבד. אפליקציית פיקוד העורף היא המקור המוסמך. הבוט נותן ערוץ נוסף בתוך Telegram/WhatsApp — מהיר יותר למי שכבר עם הטלפון פתוח שם, ותומך התראות לפי ערים שבחרת.
מתי ההתראה מגיעה?
בדרך כלל 1–3 שניות אחרי הצפירה הרשמית. הבוט מסקר את ה-API של פיקוד העורף ישירות.
האם אתם שומרים את מספר הטלפון שלי?
לא. Telegram Bot API לא חושף טלפון. ראה מדיניות פרטיות.
מה עם WhatsApp?
יש קבוצת WhatsApp המקבלת את אותן ההתראות. הלינק בדף הבית. שים לב: בוט WhatsApp רץ על whatsapp-web.js ועשוי להיות פחות יציב מטלגרם.
איך אני בוחר ערים?
פקודה /add [שם עיר] או /zones לבחירה לפי אזור. מדריך התחלה.
האם יש שעות שקט?
כן — /settings → שעות שקט. התראות ביטחוניות עדיין יישלחו (הצלת חיים), שאר הקטגוריות נשתקות.
מה זה /connect?
מאפשר לחבר את עצמך לקרוב/חבר כדי לראות את סטטוס הביטחון שלו (🛡️ OK / ⚠️ צריך עזרה) כשהוא במקלט. ראה פרטיות.
איפה המפה?
המפה נשלחת כתמונה בהודעת הערוץ. אם לא הגיעה — ייתכן שחרגת מ-Mapbox quota (הבוט עובר לטקסט בלבד).
אפשר להפעיל מופע משלי?
כן. מדריך self-host.
מה קורה אם Telegram נופל?
אין גיבוי. הבוט מאחסן ומשחזר מצב, אבל אם Telegram לא זמין — אי אפשר לשלוח. לכן אפליקציית פיקוד העורף הרשמית חייבת להישאר המקור העיקרי.
איך אני מדווח על באג?
GitHub Issues. לבעיות אבטחה: Security Advisory.
האם יש קהילה?
GitHub Discussions + קבוצת WhatsApp קהילתית.
איזה נתונים נאספים?
chat_id, ערים שבחרת, שם תצוגה אופציונלי, עיר מגורים אופציונלית. היסטוריה של 7 ימים. הפירוט המלא.
איך מוחקים אותי?
חסימת הבוט מסירה אותך. גם פנייה ב-GitHub Issue תטופל.
האם הבוט נגיש לקוראי מסך?
כן, הודעות הבוט הן טקסט. הצהרת נגישות מלאה.
למה לפעמים אני מקבל כמה פעמים את אותה התראה?
הבוט מעדכן הודעת התראה קיימת אם גל נוסף מגיע בתוך חלון של 2 דקות. אם זה קורה יותר — דווח באג.
מה זה "בדיקת שכנים"?
כ-7 דקות אחרי אירוע, הבוט שולח לחלק מהמשתמשים שאלה אם הם בדקו על שכן מבוגר/חולה. פיצ'ר חיברותי, אופט-אאוט דרך /settings.
מה עם דוברי ערבית / רוסית?
כרגע עברית בלבד. i18n מתוכנן — PRs welcome ב-GitHub.
אפשר לתרום כסף?
הפרויקט בהתנדבות. GitHub Sponsors פתוח לתרומות לתיעוד/דומיין/שרתים.
